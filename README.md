
# Calculator App

A GUI calculator using Java Swing for basic arithmetic operations.

## Features
- Addition, subtraction, multiplication, division
- Decimal support

## Setup
1. Install JDK 17.
2. Compile: `javac src/Calculator.java`
3. Run: `java -cp src Calculator`
